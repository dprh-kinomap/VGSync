VGSync – The Simple Video and GPX Sync Tool

Non-Profit Project

VGSync is a non-profit personal project aiming to provide an easy way to synchronize and edit video files with GPX data.

Key Features

  • Align your video files precisely with GPX routes for easy navigation.
  • Mark and remove unwanted video segments (e.g. MarkB / MarkE).
  • Map and chart integration to visualize routes and elevation profiles.
  • Effortless final export via ffmpeg.

Usage & Requirements

It runs on Windows or Linux (e.g., Ubuntu). You need to install VLC and ffmpeg and both must be in your PATH environment.
A stable internet connection is recommended for version checks.
